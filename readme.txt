---------- ----------
◇ 概要 ◇
少し前にMoodleで出された、時間割をExcelに入力して提出するという課題。このExcelファイルのデータを元に曜日、授業、日付の
階層フォルダを作ります。作るだけです。1年前期のみ対応です。データの分類などは行えませんのでご了承ください。
また、コマンドライン版しかありません。ご了承ください。


◇ 制作意図 ◇
C-Learningでかなりのデータを閲覧、再生したりしますよね。授業によっては古いデータが消される授業もあってダウンロード
しておきたいけど、ファイルの数が多すぎてごちゃごちゃになりそう。そんな状況を少しでも支援できたら良いなと思い、作りました。

◇ 動作条件 ◇
Windows10でのみ動作確認済みです。他のOSでの動作は不明です。


◇ インストール ◇
zipファイルを解凍すれば、そのまま利用できます。


◇ 使い方 ◇
1.「ここに時間割Excelを入れてください」に以前課題で作成したエクセルデータを入れてください。(PDFは読み込めません。)
2.uuefolder1.00.exeを実行し、指示に従う。※入力する数字は半角でお願いします。
(個人制作の為、デジタル証明書が無く警告が出ますが、問題なく使用できます。)

3.終了メッセージが表示され、終了します。
4.実行したexeのあるフォルダに「1年前期」が生成されています。デスクトップなどお好きなところに移動してお使いください。


◇ アンインストール ◇
ゴミ箱に入れればOKです。Windowsの「アプリと機能」などからアンインストールなどしなくても消えます。
